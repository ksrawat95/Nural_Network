# Nural_Network

## Project Overview
This repository contains a collection of Jupyter Notebooks demonstrating various neural network architectures and workflows. The projects range from building simple neural networks from scratch using NumPy to constructing deep Convolutional Neural Networks (CNNs) and implementing standard deep learning workflows using TensorFlow and Keras.

## Tech Stack
- **Programming Language**: Python 3
- **Libraries**: TensorFlow, Keras, NumPy, Matplotlib, Pandas
- **Environment**: Jupyter Notebook

## Key Implementation Details
- **Simple NN with Numpy.ipynb**: A clean implementation of backpropagation, forward propagation, activation functions, and gradient descent from scratch using only NumPy.
- **Keras Sequential Model.ipynb & Keras 4 Step Workflow.ipynb**: Standard implementations of multi-layer perceptrons (MLPs) using the Keras Sequential API. Demonstrates data preprocessing, model compilation, training, and evaluation.
- **CNN & CNN - Classwork.ipynb**: Convolutional Neural Networks designed for image classification tasks. Implements convolutional layers, pooling layers, dropout for regularization, and dense output layers.
- **Multi-variable equation.ipynb**: Applies neural networks/regression to model and solve multi-variable systems of equations.
